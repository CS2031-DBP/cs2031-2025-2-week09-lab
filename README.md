## Comandos Útiles

Para compilar los estilos de TailwindCSS, usar el siguiente comando:

```sh
npx tailwindcss -i ./src/input.css -o ./build/output.css --watch
```

Para compilar y observar cambios en TypeScript, usar el siguiente comando:

```sh
npx tsc --watch
```
